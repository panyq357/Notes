# 修改主目录下文件夹的中文名字为英文

首先先安装``xdg-user-dirs-gtk``。

````bash
sudo pacman -S xdg-user-dirs-gtk
````

然后临时改下语言的变量。

````bash
export LANG=en_US
````

然后用刚装的软件修改主目录下文件夹的名字。

````bash
xdg-user-dirs-gtk-update
````

运行后会弹出个窗口，问你“是否将标准文件夹更新到当前语言”，选更新即可改变主目录下文件夹的名字。

如果原来的文件夹里有东西，会同时保留中文名文件夹和英文名文件夹，把要的东西拿出来，删掉中文名的文件夹即可。
